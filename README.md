## Installation

1. Check [LazyVim](https://github.com/LazyVim/LazyVim) requirements.

2. Make a backup of your current Neovim files.

   ```bash
   # required
   mv ~/.config/nvim{,.bak}

   # optional but recommended
   mv ~/.local/share/nvim{,.bak}
   mv ~/.local/state/nvim{,.bak}
   mv ~/.cache/nvim{,.bak}
   ```

3. Clone into `$HOME/.config/nvim` directory.

   ```bash
   git clone git@github.com:enrique-mendoza/nvim.git ~/.config/nvim
   ```
